Основные этапы работы кода:

1. Загрузка и подготовка данных
Читает данные из Excel-файла 2onexo.xlsx

Удаляет строки с пропущенными значениями в целевом столбце 'ASAK'

Заполняет пропуски: медианой для числовых и модой для категориальных данных

2. Анализ распределения целевой переменной
Строит гистограмму и ящик с усами для визуализации распределения 'ASAK'

Удаляет выбросы методом межквартильного размаха (IQR)

3. Анализ корреляций
Строит тепловую карту корреляций между всеми числовыми признаками

Отбирает признаки с корреляцией > 0.3 с целевой переменной

4. Построение моделей машинного обучения
Линейная регрессия (с масштабированием признаков)

Случайный лес (без масштабирования)

Оценивает модели по метрикам MSE и R²

Визуализирует предсказания vs фактические значения

5. Анализ важности признаков
Определяет наиболее важные признаки для лучшей модели

Отбирает признаки с важностью ≥ 5%

6. Прогнозирование на исходных данных
Загружает исходные данные (включая пропущенные значения)

Заполняет пропуски аналогично обучающим данным

Делает прогнозы для всех строк исходного файла
